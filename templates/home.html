A.baumannii Clone-type Assignment Tool: ClAssT

What is ClAssT?

The 'Clone-type Assinment Tool (ClAssT)' is a free, fast and easy-to-use method for identifiying a international Clone of A.baumannii. This Tool was made to support hospital staff to fight an outbreak of multidrug resistent A.baumannii bacteria. Click here for more information.

What the Tool does

ClAssT uses exact matches to taxonomically assign (given sequence-reads of) a genome to a International Clone of A.baumannii. The identification of OXA-Families is optional. It is also possible to extend ClAssT to include more reference-genomes in the assignment-process.

How to use the Tool

Upload a .fq-file with the un-assembled or a .fasta- / .fna-file with the assembled genome by using the upload-function.

If you are using a .fq-file, you can select the number of reads that will be used (minimum is 500, maximum is 100.000). A Classification of a International Clone is usually possible with only 500 reads and is the fastest option.

The 'Quick-search' is only available for .fasta/.fna files and enables a mode where only a subset of the file-data is used for the assignment. The optional Oxa-search still uses the complete file.

The 'Include...' options allows you to select the International Clones that you want to include in your assignment-process. You can also search for OXA-Families 'on the fly' (That means that the given input reads will be checked if they contain OXA-Families). This Tool is also expandable, that means that over genomes may have been added so you can add them to your search. Added genomes are also displayed on the right.

Input files

You can upload the un-assembled reads with a .fq-file or the assembled Genome with a .fasta- or .fna-file.

Please keep in mind that the Tool uses exact matches and therefore needs high quality 'forward' Sequence-reads (e.g. Illumina Sequencing reads), otherwise a valid result is not guaranteed.

If you are using a Sequencing-technique with a high error-rate, you can use error-correction Tools before using this assignment Tool.
